# Changelog

## 0.1.0

- Initial standalone CLI for extracting paper PDF metadata, searching public bibliography sources, and selecting BibTeX.
- Added DBLP, arXiv, Crossref, OpenAlex, DOI content negotiation, and Semantic Scholar sources.
- Added `config.toml`, custom HTTP JSON sources, interactive selection, title search, and existing `.bib` update support.
- Added multi-platform binary builds.
